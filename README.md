# MapleGuildStatusOCR-KMS

메이플스토리 길드 컨텐츠 현황 화면을 읽어서  
**닉네임 / 수로 점수 / 플래그 점수**를 엑셀로 정리해 주는 반자동 도구입니다.

이 프로젝트는 **완전 자동 OCR**보다,  
**닉네임 crop을 사람이 확인하기 쉽게 잘라 주고**
**수로/플래그 점수를 안정적으로 추출하는 것**을 목표로 합니다.

---

## 주요 기능

- 여러 장의 스크린샷을 한 번에 추가 가능
- 각 이미지마다 row 개수 설정 가능
- 공식 길드원 명단 자동 수집
- 닉네임 / 수로 / 플래그 열 자동 추출
- 닉네임 row crop 자동 생성
- 닉네임 OCR + 길드원 명단 기반 자동 보정
- 수로 점수 자동 추출
- 플래그 점수 자동 추출
- 결과를 엑셀 파일(`.xlsx`)로 저장

---

## 현재 프로젝트 방향

이 도구는 다음 방향을 기준으로 설계되어 있습니다.

- **닉네임**: 자동 보정은 보조 기능, 필요 시 사람이 수기로 수정
- **수로 점수(F column)**: 최대한 정확하게 숫자만 추출
- **플래그 점수(G column)**: `0` 또는 `1000`만 추출
- **최종 목표**: 완전 자동 OCR기보다, 실사용 가능한 **반자동 정리 툴**

---

## 지원 환경

현재 가장 안정적으로 확인된 조합:

- Python **3.11**
- `paddlepaddle==2.6.2`
- `paddleocr==2.10.0`

이 조합이 아닌 경우 OCR 관련 오류가 발생할 수 있습니다.
